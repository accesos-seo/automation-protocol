# Handover Index

Índice operativo de handovers del repositorio `automation-protocol`.

## Carpetas

```text
handover/_templates/
handover/automations/
handover/ai-sessions/
handover/Registros_Historicos/
```

## Uso

```text
handover/_templates/ = plantillas reutilizables
handover/automations/ = handovers por automatización
handover/ai-sessions/ = handovers para continuidad entre sesiones IA
handover/Registros_Historicos/ = cierres históricos por fecha
```

## Regla operativa

Los handovers documentan continuidad, evidencias, estado de seguridad y próximos bloques. No deben contener secretos reales.
