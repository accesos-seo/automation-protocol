# 01 - Fuente de verdad y respaldo local

La web puede ser la fuente primaria de verdad, pero el protocolo requiere una memoria local auditada. La web dice qué existe ahora; Supabase, Storage y GitHub dicen qué se usó, cuándo se usó, cómo se interpretó y qué automatización se generó.

Cada fuente capturada debe registrar source_url, source_type, fetched_at, raw_snapshot, normalized_content, content_hash, protocol_id, phase, confidence y status.
