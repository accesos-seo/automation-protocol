# 02 - Fase 1: Intake / Specialist Approval

La Fase 1 crea o actualiza el Automation Protocol Record. En el estado actual, OpenRouter u otro proveedor de IA queda pendiente de conexión final.

Configs pendientes: AI_PROVIDER, AI_MODEL, OPENROUTER_API_KEY, OPENAI_API_KEY opcional, ANTHROPIC_API_KEY opcional. Si falta una API key obligatoria, se registra como PENDING_CONFIG.
